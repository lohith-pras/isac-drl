# ISAC-MIMO-DRL

Deep Reinforcement Learning for Integrated Sensing and Communication (ISAC) in MIMO Beamforming.

## Project Structure

| Folder      | Description                                      |
|-------------|--------------------------------------------------|
| `environment/` | Custom RL environments and wrappers            |
| `models/`      | Neural network architectures (actor, critic, etc.) |
| `training/`    | Training scripts, loops, and hyperparameters   |
| `evaluation/`  | Evaluation scripts, metrics, and visualization  |
| `utils/`       | Utility functions and helpers                  |

## Setup

This project uses [uv](https://astral.sh/blog/uv) for dependency management.

```bash
uv sync
```
